# Dehancer Brew

## How do I install these formulae?

`brew install dehancer/brew/<formula>`

Or `brew tap dehancer/brew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dehancer/brew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
